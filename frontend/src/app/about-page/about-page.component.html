About

Diese App vereinfacht die Koordination durch den Tag der offenen Tür der HTL Leonding und schafft einen guten Überblick über die Stationen, welche behandelt werden können. Die App wurde im Rahmen des Systemplanung und Projektentwicklungs - Projekts entwickelt.

Benutzerhandbuch

Funktionen

  • Leitfaden-Seite:
    Hier finden Sie einen Leitfaden, der als Orientation für den Tag der offenen Tür dient. Sie können jeden Abschnitt des Leitfadens abhaken um den Fortschritt zu sehen.
  • Tour-Seite: Hierhin gelangen Sie, wenn Sie einen Tour Abschnitt auf der Leitfaden-Seite anklicken. Hier können Sie, speziell wenn Sie einer Tour folgen nacheinander abhaken, was Sie gesehen haben. Wenn Sie alle Station der Tour erledigt und abgehakt haben, wird der Abschnitt im Leitfaden automatisch abgehakt.
  • Stationbeschreibungs-Seite: Hierhin gelangen Sie, wenn Sie auf der Tour-Seite eine Station anklicken. Hier können Sie alle Details einer Station sehen und haben die Option sich diese auf der Karte anzeigen zu lassen. Wenn Sie auf die Karte wechseln gibt es auch einen zurück Knopf um wieder auf die Stationbeschreibungs-Seite zu gelangen.
  • Karten-Seite: Zeigt eine Karte der Schule mit verschiedenen Stockwerken und Räumen. Benutzer können zwischen den Stockwerken navigieren und spezifische Räume anzeigen.
  • Navbar: Eine feste Navigationsleiste am unteren Bildschirmrand ermöglicht den schnellen Zugriff auf die Hauptbereiche der App.
